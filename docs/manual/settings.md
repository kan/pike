# 設定

設定は左下の**歯車アイコン**からシングルトンタブとして開きます。変更は自動保存され、**開いているすべての Pike ウィンドウへ即座に反映**されます。

- **言語**：日本語 / English（UI 全体が即時に切り替わります）。
- **モード**：ダーク / ライト。カラースキームとエディタテーマで **Auto** を選ぶと、このモードに追従して配色が切り替わります。
- **背景**：ウィンドウを半透明にして背後のデスクトップを透かせます。「不透明（既定）/ 透過 / アクリル」から選べます。「透過」はデスクトップがくっきり透け、「アクリル」は Windows 11 のすりガラス効果です（Windows 11 で有効）。**不透明度**スライダー（10〜100%）で透け具合を調整できます。サイドバー・パネルだけでなく、ターミナルとエディタの本文も透過します。
- **UI フォント**：アプリ画面（サイドバー・タブ・パネル・各種ダイアログ）のフォント。「システム既定」または任意のインストール済みフォントを選べます。
- **UI フォントサイズ**：アプリ画面の文字サイズ。クローム領域全体を比例スケールするため、サイズを変えてもレイアウトが崩れません（ターミナル / エディタの本文は各自の設定のまま）。
  - スライダーをドラッグ中は数値ラベルだけがサイズを先行表示し、指を離した時点で反映されます。

> **ターミナル / エディタのフォント**は、それぞれ Terminal / Editor セクションで別に設定します（UI フォントとは独立）。

## 全般

- **閉じたときトレイに最小化**：ON/OFF。ON（既定）ではウィンドウを閉じても Pike はシステムトレイに常駐し、トレイアイコンから復帰できます（終了はトレイメニューの「終了」から）。OFF ではメインウィンドウを閉じると Pike を終了します。→ [ショートカットと CLI（タスクバーとシステムトレイ）](shortcuts-and-cli.md#タスクバーとシステムトレイ)

## ターミナル

- **フォント / フォントサイズ**：ターミナルのフォント。システムのモノスペースフォントから選べます。変更は既存ターミナルに即反映されます。**この設定はエディタには影響しません**（エディタは Editor セクションで設定）。
- **カラースキーム**：6 種（Default Dark / Solarized Dark / Solarized Light / Monokai / Dracula / Nord）と **Auto（モード追従）**。Auto はアプリのダーク/ライトに合わせて Default Dark / Solarized Light を自動で選びます。凡例はターミナルフォントで表示されます。
- **選択時にコピー / 右クリックで貼り付け**：ON/OFF。
- **小さいテキストファイルをインライン展開**：→ [サイドバーパネル](panels.md#ファイル添付クリップボード--ドラッグドロップ)
- **プロセス終了時のデスクトップ通知**：ON/OFF。
- **グローバルモードの既定シェル**：グローバルモードのウィンドウで「+」を押した時に開くシェル。Windows シェル（Command Prompt / PowerShell / PowerShell 7 / Git Bash）と検出済みの WSL ディストロから選べます。非表示にしたシェルはこの選択肢からも除外されます。マシンの WSL 構成に依存するため、この設定だけは設定同期の対象外です。→ [グローバルモード](global-mode.md#ターミナルとして使う)
- **シェル一覧**：ターミナル追加の「▾」プルダウンと既定シェル選択肢に出すシェルの並び順と表示/非表示を管理します。↑↓ で並べ替え、目のアイコンで表示/非表示を切り替えます。現在の既定シェルには「デフォルト」バッジが付きます。WSL と Windows それぞれ最低 1 つは表示のまま維持されます。この設定もマシン固有のため同期対象外です。
- **ターミナルのエージェントボタン**：起動コマンドの追加・編集・並べ替え。→ [ターミナルと AI エージェント](terminal-and-agents.md#エージェント起動ボタン--プロンプト挿入)
- **ターミナルのプロンプト**：定型プロンプト（ラベルと本文）の追加・編集・並べ替え。

## エディタ

- **フォント / フォントサイズ**：エディタ本文のフォント（ターミナルとは独立）。
- **テーマ**：エディタの配色テーマ。**Auto（モード追従）** はアプリのダーク/ライトに合わせて One Dark / Default Light を自動で選びます。凡例はエディタフォントで表示されます。
- **ミニマップ**：ON/OFF。
- **折り返し（Word Wrap）**：ON/OFF。
- **タブサイズ**：2 / 4 / 8。

これらは CodeMirror に即時反映されます。

## エージェント

- **デフォルトエージェント**：新規チャット時に Claude Code / Codex のどちらを使うか、または毎回選択。
- **エージェントのデスクトップ通知**：ON/OFF。

## 設定の同期

Pike 自体には同期サービスはありませんが、**環境に依存しない設定**（フォント・配色・エディタ・ターミナルのコマンド/プロンプトなど）を 1 つの JSON ファイルに書き出せます。

- 保存先を Dropbox / OneDrive / git で同期されるフォルダ内のパスにすると、複数 PC 間で設定を共有できます。
- **書き出し（Export）** / **読み込み（Load）** を手動で行えます。起動時にも同期ファイルから自動で読み込みます。
- プロジェクト一覧やグループ、同期ファイルのパス自体、グローバルモードの既定シェルとシェル一覧は同期対象外です（環境固有のため）。

## 更新（About）

- 現在のバージョンを表示します。
- **「更新を確認」**で GitHub Releases を参照し、新しいバージョンがあれば **「更新して再起動」**で適用できます。
- 起動時にもバックグラウンドで更新を確認し、ある場合は歯車アイコンに通知ドットが付きます。
